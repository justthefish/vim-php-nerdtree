Requirements
---

 * lisp interpreter (preferrably SBCL 1.0+) for Lisp editing

Installlation
---

    cd ~
    git clone git://github.com/justthefish/vim-php-nerdtree.git .vim
    ln -s ~/.vim/.vimrc .
    cd .vim/
    git submodule update --init
    vi
    :BundleInstall
